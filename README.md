# Synera: Synergy Auto-Arena

一款基于 **Qt 6.8 + MSVC 2022** 的桌面自走棋（Auto-Battler）游戏。

> 灵感来源于《The Last Flame》和《Teamfight Tactics》。

---

## 快速开始（压缩包）

1. 右键点开压缩包进行解压
2. 找到带图标的**SYNERA**应用程序
3. 双击打开即可开玩

   
**或者：** 右键**SYNERA**应用程序 -> 点击“显示更多选项” -> 点击“发送到” -> 桌面快捷方式

就可以随时在桌面上点击开玩啦！


==**注意：** 不要把**SYNERA**应用程序直接拖到文件夹以外的地方（如桌面）！否则可能因为缺少 Qt 依赖而打不开游戏，建议在桌面创建快捷方式，请始终保持游戏本体在文件夹内，请勿随意删除文件夹内其他文件！==


## 快速开始（开发者）

### 环境要求

| 工具          | 版本                                    |
| ------------- | --------------------------------------- |
| Visual Studio | 2022（含"使用 C++ 的桌面开发"工作负载） |
| Qt            | 6.8.3 (MSVC 2022 64-bit)                |
| Windows SDK   | 10.0                                    |

### 构建 & 运行

1. **安装 Qt 6.8.3**，选择 MSVC 2022 64-bit 组件
2. **配置 Qt MSBuild**：Visual Studio 中打开菜单 `扩展 → Qt VS Tools → Qt Versions`，添加 `D:\Qt\6.8.3\msvc2022_64`
3. **打开解决方案**：双击 `AutoChessDemo.slnx`
4. **按 F5** 构建并运行

### 命令行构建

打开"VS 开发者命令提示符"：

```bash
msbuild AutoChessDemo.slnx /p:Configuration=Debug /p:Platform=x64
```

Release 构建：

```bash
msbuild AutoChessDemo.slnx /p:Configuration=Release /p:Platform=x64
```

## 游戏玩法

### 界面介绍

游戏主窗口分为以下几个区域：

```
┌──────────────────────────────────────────────┐
│  棋盘区域（8×8 网格）                         │
│  ┌─────────────────────────────────────┐      │
│  │ 上半区（敌方半场 — 红色）           │      │
│  │                                     │      │
│  ├─────────────────────────────────────┤      │
│  │ 下半区（我方半场 — 绿色）           │      │
│  │                                     │      │
│  └─────────────────────────────────────┘      │
├──────────────────────┬───────────────────────┤
│   备战区（8 格）      │   状态信息             │
│   [ ][ ][ ][ ][ ][ ][ ]│   金币: 10            │
│                        │   波数: 第 1 波       │
│                        │   生命: ♥ 100         │
│                        │   羁绊: 人类(2) 等    │
├────────────────────────┤                       │
│   商店（5 张卡牌）     │                       │
│   [步兵][弓箭手]...    │                       │
│   [刷新商店] [开始战斗]│                       │
└────────────────────────┴───────────────────────┘
```

- **棋盘** — 8×8 网格，上半区为敌方半场（红色），下半区为我方半场（绿色）。战斗时单位自动寻路攻击敌人
- **备战区** — 位于棋盘下方，共 8 格，用于暂存未上场的单位
- **商店** — 每回合刷新 5 个随机单位，点击购买加入备战区
- **状态栏** — 显示当前金币、波数、玩家生命值及已激活的羁绊信息

### 基本流程

游戏共 **15 波** PvE 回合，击败全部 15 波即为通关。第 15 波为 Boss 关，Boss 属性极高（约 3000 HP / 180 ATK）。每回合分为两个阶段：

#### 1. 准备阶段

在商店中浏览本回合刷新的 5 个单位，点击心仪的单位消耗金币购买。购买后单位会进入备战区，从备战区拖拽单位到棋盘我方半场（绿色区域）即可部署上场。部署好阵容后点击「开始战斗」进入战斗阶段。

#### 2. 战斗阶段

己方单位与敌方 AI 单位自动交战。单位会根据敌我位置自动寻路、攻击和施放技能。战斗结束后，胜利可获得金币奖励（随波数递增），失败则获得 5 金币补偿，并扣除 `10 + 波数` 点生命值。生命归零则游戏结束。

> **提示**：合理搭配羁绊、合星提升单位品质，是通关的关键！

### 操作指南

| 操作         | 方式                                                     |
| ------------ | -------------------------------------------------------- |
| 购买单位     | 点击商店中的单位卡片，消耗金币购买                       |
| 部署上场     | 从**备战区**拖拽单位到棋盘我方半场（绿色区域）           |
| 撤回备战区   | 从**棋盘**拖拽我方单位到备战区                           |
| 出售单位     | 点击单位后在右侧详情面板点击「出售」按钮，或右键点击**备战区**单位 |
| 刷新商店     | 点击「刷新商店」按钮（前 2 次免费，之后逐次递增 1 金币） |
| 开始战斗     | 部署完成后点击「开始战斗」                               |

### 单位介绍

游戏中有 **9 种可招募单位**，分为 1~3 费三个梯度，费用越高基础属性越强：

| 费用 | 单位     | 生命 | 攻击 | 射程 | 羁绊             | 定位       |
| ---- | -------- | ---- | ---- | ---- | ---------------- | ---------- |
| 1★   | **步兵** | 300  | 25   | 1    | 人类、战士       | 前排肉盾   |
| 1★   | **弓箭手** | 200 | 30   | 3    | 精灵、远程       | 后排输出   |
| 1★   | **侍从** | 250  | 20   | 1    | 人类、侍从       | 辅助前排   |
| 2★   | **法师** | 250  | 40   | 2    | 人类、法师       | 法术输出   |
| 2★   | **骑士** | 500  | 20   | 1    | 人类、骑士       | 重型坦克   |
| 2★   | **治疗师** | 300 | 15   | 2    | 精灵、治疗       | 续航辅助   |
| 3★   | **刺客** | 200  | 50   | 1    | 精灵、刺客       | 高爆发近战 |
| 3★   | **狂战士** | 400 | 45   | 1    | 兽人、战士       | 主力输出   |
| 3★   | **狙击手** | 250 | 55   | 4    | 精灵、远程       | 超远程炮台 |

第 15 波 Boss 战还会出现特殊 Boss 单位（恶魔/Boss 标签，3★ 属性），属性极高。

单位每回合自动恢复满血量，战斗中的法力值也重新从 0 开始积攒，法力攒满后自动施放技能。

**星级倍率已加强**：2★ = 基础 × **2.0**，3★ = 基础 × **3.0**，高星单位战斗力碾压低星。

### 经济系统

- 胜利金币奖励：**当前波数 × 2 + 8**（随波数递增，第15波胜利获得 38 金币）
- 失败金币补偿：**当前波数 × 2 + 5**（保底收益，逆风局也能逐步积累）
- 失败扣血：**当前波数 × 4**（HP 归零则游戏结束）
- HP 恢复：胜利且 HP≠100 时，恢复 **当前波数 × 2** 点生命值（有翻盘可能）
- 出售单位返还金币：费用 × 星级倍率（1★×1、2★×3、3★×9）
- 金币用于购买单位和刷新商店（前 2 次免费，之后逐次递增 1 金币）

### 合星系统

- **3 个同名 1★ 单位 → 1 个 2★ 单位**（属性 ×2.0，大幅提升）
- **3 个同名 2★ 单位 → 1 个 3★ 单位**（属性 ×3.0，质的飞跃）
- 准备阶段开始时会自动检测并合星
- 高星级单位战斗力远超低星级，合理合星是取胜关键

**星级倍率**：1★ = 1.0x（基础）→ 2★ = **2.0x** → 3★ = **3.0x**

### 羁绊系统

每个单位拥有**羁绊标签**（如"人类"、"战士"、"精灵"、"法师"等）。场上同名羁绊数量达到阈值时激活队伍加成，**战斗前自动为所有己方单位施加属性增益**：

- **≥2 个同羁绊单位** → 激活一档（Ⅰ）
- **≥4 个同羁绊单位** → 激活二档（Ⅱ，效果更强）

| 羁绊 | 一档效果 | 二档效果 | 受益单位 |
|------|---------|---------|---------|
| 人类 | ATK+10% | ATK+25% | 步兵、侍从、法师、骑士 |
| 战士 | HP+10% | HP+25% | 步兵、狂战士 |
| 兽人 | ATK+12% | ATK+25% | 狂战士 |
| 精灵 | ATK+8% | ATK+20% | 弓箭手、治疗师、刺客、狙击手 |
| 远程 | 射程+1 | 射程+2 | 弓箭手、狙击手 |
| 法师 | ATK+12% | ATK+25% | 法师 |
| 骑士 | HP+12% | HP+25% | 骑士 |
| 治疗 | ATK+8%, HP+8% | ATK+15%, HP+15% | 治疗师 |
| 刺客 | ATK+15% | ATK+30% | 刺客 |
| 侍从 | HP+10% | HP+20% | 侍从 |

一个单位同时拥有多个羁绊标签时，可叠加享受多个加成效果。例如 步兵（人类+战士）在有 ≥2 人类和 ≥2 战士的阵容中，同时获得 ATK+10% 和 HP+10%。

合理搭配不同羁绊可以大幅提升阵容强度，是通关第 15 波 Boss 的关键策略。

### 装备系统

每 4-6 回合胜利可掉落装备，装备可提升单位属性（如攻击力 +15、生命 +150 等）。低级装备可以合成高级装备。

## 技术栈

| 层面     | 技术                                                     |
| -------- | -------------------------------------------------------- |
| 语言     | C++17                                                    |
| UI 框架  | Qt 6.8 (Widgets + QPainter)                              |
| 构建工具 | MSBuild / Visual Studio 2022                             |
| 音效     | Windows PlaySound API (winmm)                            |
| 寻路     | BFS 网格寻路                                             |
| 战斗     | 状态机 FSM（Idle → Moving → Attacking → Casting → Dead） |

### 核心设计

- **纯逻辑与 UI 分离**：`GameCore/` 下所有代码不依赖 Qt，可在无 GUI 环境下复用
- **自绘界面**：所有 UI 组件使用 QPainter 自定义绘制，不依赖 QML
- **游戏循环**：10ms 定时器驱动（~100fps），`GameManager::Update()` 驱动战斗逻辑

## 游戏平衡性验证（数值报告）

### 经济系统
- 全胜总收入（含初始 10g）：**460 金币**（胜利 formula 下调，鼓励精打细算）
- 失败补偿：每波 `波数×2+5`，保证逆风局经济不崩盘
- 商店刷新：前 2 次免费，第 3 次起每次递增 1 金币，降低前期刷新成本

### 商店概率
| 商店等级 | 1 费概率 | 2 费概率 | 3 费概率 |
|---------|---------|---------|---------|
| 1（第1波） | 100% | 0% | 0% |
| 2（第2-3波） | 65% | 30% | 5% |
| 3（第4-5波） | 40% | 35% | 25% |
| 4（第6-7波） | 20% | 35% | 45% |
| 5（第8-9波） | 10% | 30% | 60% |
| 6+（第10波起） | 5% | 20% | **75%** |

中后期商店大量出现 3 费单位，凑 3★ 核心输出不再困难。

### 敌人数值曲线
| 波次 | 倍率 | 敌方单位 | 总 HP 池 | 难度 |
|-----|------|---------|---------|------|
| 1 | 1.00x | 2 步兵 | 600 | 新手 |
| 5 | 1.32x | 2 骑士 + 2 法师 + 治疗 | 2376 | 渐入 |
| 10 | 1.86x | 3 狂战 + 2 狙击 + 2 法师 + 治疗 | 4942 | 中级 |
| 13 | 2.33x | 混编 10 单位 | 7926 | 困难 |
| 15 | 2.52x | Boss + 10 精英 | ~8200 | **Boss** |

### 星级属性提升
| 星级 | 倍率 | 举例（步兵） | 举例（狂战士） |
|-----|------|------------|--------------|
| 1★ | 1.0x | 300HP / 25ATK | 400HP / 45ATK |
| 2★ | **2.0x** | 600HP / 50ATK | 800HP / 90ATK |
| 3★ | **3.0x** | 900HP / 75ATK | 1200HP / 135ATK |

### 羁绊加成效果
羁绊系统在战斗前自动激活，为场上单位叠加属性 Buff，详细效果见上方「羁绊系统」章节。

### 总体评价
游戏可以通关，但需要合理策略：围绕羁绊组队、优先合星、搭配前后排阵容。对新手有一定挑战，但不会数值碾压。第 15 波 Boss 战是终极考验。 

## 打包发布

1. 以 **Release** 方式构建：`msbuild AutoChessDemo.slnx /p:Configuration=Release /p:Platform=x64`
2. 输出位于 `x64/Release/SYNERA.exe`
3. 运行 windeployqt 收集依赖：
   ```bash
   cd x64/Release
   windeployqt SYNERA.exe
   ```
4. 将整个文件夹压缩，发送给好友即可
5. 如果对方缺少 VC++ DLL，安装 [VC++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)
